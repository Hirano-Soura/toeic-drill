# TOEIC 学習計画 v2（2026-07-16 改訂）

## 改訂のきっかけ: IP試験結果

- **L315 / R165 / Total 480**（2026年7月受験、IPオンライン=CAT方式）
- 診断: Lは平均以上で悪くない。Rが極端に低く、原因は (1) 語彙不足、(2) Part 6文挿入・Part 7語彙言い換えなど未習形式、(3) 読解速度。
- IPオンラインの特性: 先読み不可のリスニング、前半正答率で後半難易度が変わるCAT方式 → 「基礎問題を確実に取る」価値が高い。

## 学習の仕組み（2本立て）

### 1. 単語ドリル（毎日 8:45、タスク: daily-toeic-vocab）
- 毎日15語の新出語（Phase 1: 600点レベル → Phase 2: 730点レベル）
- データ: `単語データ/VOCAB_YYYY-MM-DD.js`
- 閲覧: **`単語ドリル.html`** — 一覧・フラッシュカード・4択クイズ。クイズの「復習セット」は今日+1/3/7/14/30日前の語を自動で出題（間隔反復）。間違えた語は自動で苦手登録。

### 2. 本編ドリル（水・金・日 9時、タスク: daily-toeic-drill）
曜日でトラックを固定し、Rの未習形式を集中的に埋める。

- **水 = トラックG**: 残りの文法 → Part 6（時制・接続表現・**文挿入**）→ 語法 → CAT戦略
- **金 = トラックR**: Part 7（主旨・詳細・**語彙言い換え**・NOT・推測・複数文書・速読）
- **日 = トラックL**: リスニングスクリプト読解（音声なし）。先読みなし形式への対応(L7)を含む

- データ: `演習データ/TOEIC_YYYY-MM-DD.js`（JSON内容をJSでラップした形式）
- 閲覧: **`演習ビューア.html`** — 日付タブ切替、クリック解答、解説表示
- 復習: 各回に「1回前・3回前・7回前」のトピックからの復習問題を含む
- トピック一覧と進捗の分母は `_curriculum.md`（全40、うち9消化済み）

## 運用メモ

- 旧形式の9回分（文法No.01〜09）は `デイリー演習.html` でそのまま復習できます。
- アプリ（Claude）が起動していない時間帯のタスクは、次回起動時に実行されます。
- トラックを消化し終えたら、各曜日は既出トピック横断の「MIX実戦回」に移行します。
